Classic Mac Theme Font Files
============================

These WOFF2 files are PLACEHOLDERS that should be replaced with actual font files.
The placeholder files have valid WOFF2 headers but contain no glyph data.
The Classic theme's font fallback chains in classic-tokens.css will activate
gracefully when these placeholders fail to load:

  --gooey-font-family: "ChicagoFLF", "Chicago", Geneva, sans-serif;
  --gooey-font-family-mono: "Monaco", monospace;

   - Alternative: System Monaco font (available on macOS), or use monospace fallback

Replacement Instructions:
-------------------------
1. Obtain the actual font files (TTF/OTF/WOFF2)
2. If not already WOFF2, convert using: woff2_compress fontfile.ttf
3. Replace the placeholder files in this directory
4. The font-faces attribute on <gooey-theme> handles loading via FontFace API
